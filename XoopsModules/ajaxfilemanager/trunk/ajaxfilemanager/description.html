What is this module

This is a new Addresses module.
You can add/edit/manage extra fields for locations.
With this version you can store any kind of iamge/file (image, audio, video, text, pdf, ...)
With this version uploaded images will be automatically resized if they are bigger than max category sizes.
With this version you can install, activate or deactivate an extra Xoops tag, [loc] tag. (cat_id, loc_id, id, ...)
With this version you have an enhanced version of popup image/file manager, "imagemanager.php" script.
TO DO Module support 'TAG' module tags system.
TO DO Module support 'SITEMAP' module sitemap system.
TO DO Module support 'Waiting Contents Extensible' module system.
TO DO Module support 'What's New' module system.
