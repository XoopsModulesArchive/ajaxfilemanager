Description

What is this module

This module add an Ajax File Manager to Xoops.

This module install a new button in 'dhtmltextarea' elements and add 2 extra extensions (Video to insert .flv videos, Gmap to insert .kml generated google maps).

As resource for module developers I created 4 new formelements objects to Xoops framework... you can find them in /class directory, try them in your modules.

  • 'FormAjaxFileManager' and 'FormMultipleAjaxFileManager'
  • 'FormXoopsImage' and 'FormMultipleXoopsImage'
